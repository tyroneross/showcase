---
name: tag
description: Add/remove tags, update title, feature, component, or star a capture
allowed-tools: ["showcase"]
---

Update metadata on a capture using the Showcase `tag` tool.

The user should provide a capture ID and the changes they want (add tags, remove tags, set title/feature/component, star/unstar). If they reference a capture by description rather than ID, use `find` first to locate it.
